Setting Up a Request

Element Description Usage
Name The name of the playlist. MySongs, Christmas, etc.
Songs Specify the number of songs you want
in the playlist. Select 'no limit' for no limit.		 Positive integers: 1, 5, 22
Generator Choose the type of generator you want:
  • Simple: Songs that meet all criteria.
  • Majority: Songs that meet a majority of the criteria.
  • Tolerance: Songs that can fail a specified number of criteria.
 --
Rules Choose a trait to maximize or minimize. Least plays, most requested
Advanced Add criteria of the form "[trait] [operator] [value]". Title contains Christmas, Artist contains Elvis
Clear Clears the form. --
Create Creates and stores the playlist. --

Troubleshooting

Error Cause Resolution
Playlist name already exists. Playlist name already exists. Change the playlist name or delete existing playlist.
Song generation failed. Database failure or incorrect rules. Check rules or restart.
Invalid song or tolerance number entered. Song or tolerance value is incorrect. Change song or tolerance value to a correct number.
There are no songs that meet your criteria. There are no songs that meet your criteria. Add songs that meet your criteria or change your criteria.

To compile...
Windows: run project.bat from the command line in the src directory with an argument for the directory where you want the compiled code to go
ex. project C:\Users\Alicia\Scherzo
This will compile the project to the given directory

Not Windows:

To run...
Windows:run projectRun.bat from the directory with the compiled code, lib, imgs, and localizatonEN.properties file.

Not Windows:

Make sure that the libs and imgs folders are in the directory you want to compile to and run from.
